Formatações PT2

Textos grandes e textos pequenos

Os textos grandes são editados no CSS, mas a sua antiga tag era BIG.
Porém, a tag de textos menores continua sendo utilizada, é chamada de SMALL.

Texto excluído e Texto inserido

O texto excluído deve ser LIDO mas não considerado. Sua tag é <del>
Exemplo: Lorem ipsum, dolor sit amet consectetur adipisicing elit. Similique dicta vero repellat! Magni alias nulla dolore distinctio nemo, commodi officia voluptas nobis voluptatibus, tenetur repudiandae nihil. Ullam facilis sequi sunt?

Já o texto inserido DEVE ser lido com atenção. Sua tag é <ins>
Exemplo: Lorem ipsum dolor sit amet consectetur adipisicing elit. Soluta dicta sed accusantium fugiat aliquam, asperiores quod doloremque aliquid dolorum voluptas nulla nihil unde facilis? Temporibus neque porro dolore. Vero, quo.

Textos sobrescrito e subscrito

No texto sobrescrito a tag utilizada é SUP.
Exemplo:x+10+9.

No texto subscrito a tag utilizada é SUB
Exemplo:H2O.

Códigos fonte

O código fonte tem como função facilitar a leitura de um documento com a fonte monoespaçada, sua tag em questão é CODE.
Exemplo: Veja aqui o document_type-html

A também a tag PRE que mantém o texto pré-formatado.

Citações simples e completas

Quando a citação tem semântica e é curta a tag utilizada é Q. Exemplo: O computador é um burro muito rápido

Já as citações completas são as que encontramos nos livros, ou trechos de reportagens, jornais etc. É possível colocar o link para acessar o livro citado, A tag utilizada é BLOCKNOTE, para adicionar o link do conteúdo junto com as citações dentro da tag é necessário colocar CITE.
Exemplo: Um trecho do livro As veias abertas da américa latina:

"A chuva que irriga os centros do poder imperialista afoga os vastos subúrbios do sistema. Do mesmo modo, e simetricamente, o bem-estar de nossas classes dominantes – dominantes para dentro, dominadas de fora – é a maldição de nossas multidões, condenadas a uma vida de bestas de carga".
