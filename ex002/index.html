Parágrafos, quebra linhas e emoji

Pra criar um parágrafo é necessário colocar o texto dentro de um para de tags <p>.

Quando necessário quebrar um parágrafo usa-se a tag <br>.

Alguns simbolos podem ser adicionados com tags, alguns exemplos: ®©™Δ entre outras.

Emoji

Para usar emojis em sites, é importande usar os códigos dos emojis encontrados em vários sites, o principal deles e o EMOJIPEDIA.
&#x+(código do emoji).

Exemplo: 🇧🇷 💛
