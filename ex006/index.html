Formatações no HTML

Quando trabalhamos com HTML é importante termos em mente que, HTML é uma linguagem de CONTEÚDO, CSS é uma linguagem para o DESINGN de um site, e JAVASCRIPT cuida da INTERATIVIDADE de um site.

Nesse sentindo as tags usadas no conteúdo, tem como regra agir com Semântica, ou seja, precisa ter um significado.

Observação: Algumas tags do HTML4 se tornaram obsoletas no HTML5, isso é comum no meio de linguagens, por isso é importante sempre pesquisar quais tags ficaram obsoletas.

Principais Formatações

Negrito e Destaque

A antiga tag para negrito era B, porém essa tag não é usada pois não a semântica.
A nova tag com semântica é STRONG, exemplo: Nesse texto existe uma parte em negrito, essa é a parte!

Itálico e Enfâse

A tag para Itálico é I, porém não a semântica.
Já para dar enfâse, é usada a tag EM com semântica.

Marca textos

A tag utlizada é MARK, mas dentro da tag é possível mudar a cor do marca texto, adicionando STYLE dentro da tag MARK.
Exemplo: Eu sou jovem e tenho 19 anos, sou apenas um bebê.
