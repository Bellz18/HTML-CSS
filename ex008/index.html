Listas com HTML

Listas ordenadas/ Ordered list

Onde a ordem dos itens tem impotância.
Exemplo:Receita de um bolo de saquinho

  1. 3 Ovos
  2. 150ml de leite
  3. 3 colheres (sopa) de manteiga

Listas não ordenadas/ Unordered list

Onde a ordem dos itens não tem importância.
Exemplo: Lista de compras.

  • Pão
  • Queijo
  • Leite
  • Candida
  • Sabão em pó

Listas Mistas

É a mistura das listas ordenadas e não ordenadas.
Exemplo: Lista das músicas/bandas favoritas.

  1. The beatles
    1. In my life
    2. Blackbird
    3. Eigth days week
  2. Liniker
    1. Zero
    2. Vitoriosa
    3. Baby

As tags usadas são: DL- definition listas, DT- Definition term, DD- Definition description.
